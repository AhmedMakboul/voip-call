# Privacy Policy

This application uses Facebook Messenger to collect user-provided information.

## Information We Collect
- Messages sent to the Facebook Page
- User responses to questions

## How We Use Information
- To process user responses
- To store answers for internal use

## Data Storage
- Data is stored securely and not shared with third parties.

## Data Retention
- Data is retained only as long as necessary for the stated purpose.

## User Rights
- Users may request deletion of their data by contacting the page administrator.
